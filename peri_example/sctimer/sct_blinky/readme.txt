State Configurable Timer (SCT) blinky Example
==========================================

Example Description
-------------------
This example, uses the SCT to generate PWM signal. The PWM is outputed by P0_27 which connects to board led.

Special Connection Requirements
-------------------------------

Board [NXP_LPCXPRESSO_824]:

No special connection is needed.
